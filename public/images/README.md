# Logo DM Kuliner

Taruh file logo di folder ini dengan nama **persis**:

    dm-kuliner-logo.png

Begitu file ada, logo otomatis muncul di:
- Halaman login
- Header layar kasir
- Panel admin (topbar) & portal vendor
- Header dashboard

Kalau file belum ada, aplikasi menampilkan badge "DM" coklat sebagai cadangan.

Saran: PNG latar transparan, tinggi minimal ~200px, rasio mendekati logo asli.
